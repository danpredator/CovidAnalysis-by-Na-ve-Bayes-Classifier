Project on Predicting the Presence of Corona Virus based on Region

Project Overview¶

The Corona virus dataset contains the data related to places which are effected by the virus , the main objective of this dataset is to create a model which can help us in predicition the Presence of Corona Virus based on given Latitude and Longitude . For this classification we have used Naive Bayes Classifier under Gaussian model.¶

Data Information The dataset contains- 5822 records samples of different countries all over the world.

Attribute Information:

  1. Province/State
  2. Country/Region
  3. Latitude
  4. Longitude
  5. Date
  6. Confirmed
  7. Deaths
  8. Recovered
    --- If The case in confirmed then that Region in Effected

Step 5- Cross Validation Score

What this does is divide the entire datasets to multiple blocks of Sub-data, then all this Sub-data is trainned and tested under already built model. Along with the multiple train we get multiple accuracies. Thus mean so all the accuracies gives Final Accuracy.¶

For this cross validation we import cross_val_scorce which is inside sklearn.model_selection¶

In [14]:
from sklearn.model_selection import cross_val_score
accuracies=cross_val_score(estimator=classifier,X=X_train,y=Y_train,cv=10) 
print("Final Accuracy is :",accuracies.mean()*100)

Final Accuracy is : 72.44976694817481
